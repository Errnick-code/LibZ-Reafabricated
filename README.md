# LibZ Refabricated

**Unofficial port of [LibZ](https://modrinth.com/mod/libz) by [Globox_Z](https://modrinth.com/user/Globox1997) — ported to Minecraft 1.21.11 with Mojang Mappings (Fabric).**

All credit for the original mod goes to **Globox_Z**.

Required dependency for **LevelZ Refabricated**.

![1](https://cdn.modrinth.com/data/cached_images/7c7e053aced60c79f32739e26758cdc5314aae08.jpeg)

## What's changed in this port
- Ported to Minecraft 1.21.11
- Migrated from Yarn to Mojang Mappings

## Requirements
- [Fabric API](https://modrinth.com/mod/fabric-api)
- [Cloth Config API](https://modrinth.com/mod/cloth-config)
